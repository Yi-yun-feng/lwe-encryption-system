🔑 RSA 算法介绍

非对称加密 不抗量子 1977 年提出

📌 基本原理

RSA 是最早实用的公钥加密算法，基于大整数分解的数学难题。安全性依赖于：给定一个大合数 n = p × q（p、q 为大质数），在计算上难以分解出 p 和 q。

📐 数学公式

密钥生成：

1. 选择两个大质数 p, q

2. 计算 n = p × q

3. 计算 φ(n) = (p-1)(q-1)

4. 选择 e (1 < e < φ(n), gcd(e,φ(n))=1)

5. 计算 d = e⁻¹ mod φ(n)

公钥：(e, n)    私钥：(d, n)


加密： c = m^e mod n

解密： m = c^d mod n

🌊 LWE 算法介绍

格基加密 抗量子 2005 年提出

📌 基本原理

LWE (Learning With Errors) 是基于格问题的密码学原语。核心思想是：在高维空间中，给定一些带有小噪声的线性方程，恢复秘密向量在计算上是困难的。

📐 数学公式

参数：

n = 维度，q = 模数，χ = 噪声分布（通常为离散高斯）


密钥生成：

1. 随机生成矩阵 A ∈ Z_q^(n×n)

2. 随机生成秘密向量 s ∈ Z_q^n

3. 生成噪声向量 e ← χ^n

4. 计算 b = A·s + e mod q

公钥：(A, b)    私钥：s


加密（明文 m ∈ {0,1}）：

1. 随机生成 r ∈ {0,1}^n

2. u = A^T·r mod q

3. v = b^T·r + m·⌊q/2⌋ mod q

密文：(u, v)


解密：

m = round((v - s^T·u) / ⌊q/2⌋) mod 2

📊 优劣势总结

🔑 RSA 优势

  • 成熟稳定，已使用 40+ 年
  • 实现简单，库支持完善
  • 密钥相对较小
  • 加密解密速度快
  • 广泛部署于现有系统

🔑 RSA 劣势

  • 量子计算机可破解 (Shor 算法)
  • 需要大密钥保证安全
  • 无可证明安全性
  • 密钥生成较慢

🌊 LWE 优势

  • 抗量子计算机攻击
  • 可证明安全性
  • NIST 后量子标准候选
  • 支持全同态加密 (FHE)
  • 灵活的参数选择

🌊 LWE 劣势

  • 密钥和密文较大
  • 计算开销较大
  • 实现复杂度高
  • 标准化仍在进行中

🎯 结论与建议

当前阶段：RSA 仍然是主流选择，但应开始规划向后量子密码的迁移。

长期规划：对于需要长期安全性的数据，建议采用 RSA+LWE 混合加密方案。

研究热点：LWE 及其变体 (RLWE, MLWE) 是后量子密码学最活跃的研究方向。
