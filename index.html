🔑 RSA 算法研究与实验

非对称加密 不抗量子 Rivest-Shamir-Adleman, 1977

📌 算法原理

RSA 是最早实用化的公钥加密算法，其安全性基于大整数分解问题的计算困难性。给定一个大合数 n = p × q（p、q 为大质数），在经典计算机上分解出 p 和 q 需要指数级时间。

【密钥生成】

1. 随机选择两个大质数 p, q（通常各 1024 位以上）

2. 计算模数 n = p × q

3. 计算欧拉函数 φ(n) = (p-1)(q-1)

4. 选择公钥指数 e，满足 1 < e < φ(n) 且 gcd(e, φ(n)) = 1（常用 65537）

5. 计算私钥指数 d = e⁻¹ mod φ(n)

公钥：(e, n)     私钥：(d, n)


【加密】 c = m^e mod n

【解密】 m = c^d mod n

🌊 LWE 算法研究与实验

格基加密 抗量子 Regev, 2005

📌 算法原理

LWE (Learning With Errors) 是基于格问题的密码学原语。核心思想是：在高维格空间中，给定一组带有小噪声的线性方程，恢复秘密向量在计算上是困难的。即使使用量子计算机，目前也没有已知的有效攻击算法。

【参数设置】

n = 维度 (通常 512-4096), q = 模数 (素数), χ = 离散高斯噪声分布


【密钥生成】

1. 随机生成矩阵 A ∈ Z_q^(m×n)

2. 随机生成秘密向量 s ∈ Z_q^n

3. 从噪声分布 χ 采样误差向量 e ∈ Z_q^m

4. 计算 b = A·s + e mod q

公钥：(A, b)     私钥：s


【加密】 (明文 m ∈ {0,1})

1. 随机生成 r ∈ {0,1}^m

2. 计算 u = A^T·r mod q

3. 计算 v = b^T·r + m·⌊q/2⌋ mod q

密文：(u, v)


【解密】

计算 m' = v - s^T·u mod q

若 m' 接近 0 则输出 0，若 m' 接近 q/2 则输出 1

📈 优劣势深度分析

🔑 RSA 优势

  • 技术成熟，经过 40+ 年密码分析验证
  • 实现简单，所有主流密码库均支持
  • 密钥和密文相对较小，传输开销低
  • 加解密速度快，适合高吞吐场景
  • 现有基础设施广泛部署，兼容性好

🔑 RSA 劣势

  • 量子计算机可用 Shor 算法破解 (2048 位 RSA 约需 4000 量子比特)
  • 为维持安全性需不断增加密钥长度，性能下降
  • 无严格的安全性归约证明
  • 密钥生成较慢 (需寻找大质数)

🌊 LWE 优势

  • 抗量子计算机攻击，面向未来安全
  • 有严格的安全性归约证明 (归约到 GapSVP)
  • NIST 后量子密码标准已发布 (ML-KEM/Kyber)
  • 支持高级密码学原语 (全同态加密、属性基加密等)
  • 参数选择灵活，可调节安全等级

🌊 LWE 劣势

  • 密钥和密文尺寸大，带宽和存储开销高
  • 计算复杂度较高，加解密速度慢
  • 实现复杂，需处理噪声、参数选择等问题
  • 标准化和部署仍在进行中，生态不完善

🎯 研究结论与迁移建议

📌 核心结论

  • 短期 (1-3 年)：RSA 仍可安全使用，但应开始评估后量子迁移方案
  • 中期 (3-7 年)：采用混合加密方案 (RSA + LWE)，兼顾兼容性和未来安全性
  • 长期 (7 年以上)：全面迁移到后量子密码标准 (ML-KEM/Kyber, ML-DSA/Dilithium)
  • 高安全场景：对于需要长期保密的数据 (如国家机密、医疗记录)，应立即采用后量子加密
