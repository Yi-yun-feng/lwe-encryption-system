🔑 RSA 算法研究与实验

非对称加密 不抗量子 Rivest-Shamir-Adleman, 1977

📌 算法原理

RSA 是最早实用化的公钥加密算法，其安全性基于大整数分解问题的计算困难性。

【密钥生成】

1. 选择两个大质数 p, q

2. 计算 n = p × q

3. 计算 φ(n) = (p-1)(q-1)

4. 选择 e (1 < e < φ(n), gcd(e,φ(n))=1)

5. 计算 d = e⁻¹ mod φ(n)

公钥：(e, n)     私钥：(d, n)


【加密】 c = m^e mod n

【解密】 m = c^d mod n
