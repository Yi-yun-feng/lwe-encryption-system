🔑 RSA 算法实验（大创项目版）

非对称加密 不抗量子（Shor算法） 密钥长度：512/1024/2048位

1. 算法核心原理

安全性基础： 大整数分解问题（IFP）—— 已知 n=pq（p,q为大质数），求解 p/q 是计算困难的

密钥生成步骤：

1. 随机选择两个不相等的大质数 p, q（建议长度≥512位）

2. 计算模数 n = p × q（公钥/私钥共享模数）

3. 计算欧拉函数 φ(n) = (p-1)(q-1)（私钥核心参数）

4. 选择加密指数 e：1 < e < φ(n) 且 gcd(e,φ(n))=1（通常取 65537）

5. 计算解密指数 d：d = e⁻¹ mod φ(n)（私钥核心参数）

加密： c = m^e mod n（m为明文，c为密文）

解密： m = c^d mod n

5. 实验结论思考（大创报告参考）

• 密钥长度对 RSA 安全性的影响：越长越安全，但性能损耗呈指数级增长

• 量子计算威胁：Shor 算法可在 O((log n)³) 时间内分解 n，直接破解 RSA

• 应用场景限制：RSA 适合短消息加密（如密钥分发），不适合大数据加密

• 对比预期：LWE 算法在抗量子性上具有绝对优势，但参数选择（n/q/σ）对性能影响显著
