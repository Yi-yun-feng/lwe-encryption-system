# LWE 加密系统 | RSA+LWE 密码学算法对比研究平台
基于 LWE（Learning With Errors）的后量子加密算法实现，同时集成传统RSA公钥加密算法，提供交互式实验与多维度对比分析平台。

## 项目地址
- 在线演示平台：https://yi-meng.top/lwe-encryption-system/
- 代码仓库：https://github.com/Yi-yun-feng/lwe-encryption-system

## 项目特色
- ✅ **交互式实验环境**：一键生成密钥、加解密、实时查看运算结果与耗时
- ✅ **双算法对比**：传统RSA与后量子LWE算法并行演示，直观对比差异
- ✅ **完整LWE-RABE实现**：基于格的可撤销属性基加密方案，支持属性基访问控制与用户撤销
- ✅ **实验过程记录**：自动记录所有操作，支持一键清空与导出
- ✅ **多维度分析**：从数学基础、抗量子能力、性能、密钥尺寸等维度全面对比
- ✅ **纯前端实现**：无需后端服务，直接在浏览器中运行，部署简单
